HOW TO USE:
===========

1. DIRECT NAVIGATION:
   Visit: http://localhost:3000/users
   (The page is already set up at app/users/page.tsx)

2. INTEGRATE WITH EXISTING NAV:
   Add to your navigation or home page:

   ```tsx
   import { UsersPage } from '@/components/UsersPage'
   
   // Use it directly
   <UsersPage onBack={() => router.back()} />
   ```

3. ADD TO BOTTOM NAVBAR:
   To add users tab to existing navigation, update the tabs array:

   ```tsx
   const tabs = [
     { id: 'home', icon: faHome, label: 'Home' },
     { id: 'stats', icon: faChartBar, label: 'Analytics' },
     { id: 'users', icon: faUsers, label: 'Players' },  // NEW
     { id: 'leaderboard', icon: faTrophy, label: 'Champions' }
   ];
   ```
